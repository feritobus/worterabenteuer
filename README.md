# WörterAbenteuer

A Flutter app for Spanish-speaking kids (ages 7-12) learning German vocabulary, with a reward system that converts study time into approved screen time.

## Features

- **Family-based profiles** — one parent account, multiple kid profiles per family
- **OCR import** — point the camera at a vocab sheet or Antón screenshot, get an editable list of German/Spanish pairs
- **Four study modes** with different point values:
  - Flash Cards (+1 pt) — view the translation
  - Keyboard (+2 pts) — type the German word
  - Handwriting (+7 pts) — draw it; ML Kit Digital Ink recognizes German; dominates the word
  - Voice (+7 pts bonus) — speech-to-text; unlocked after a perfect round
- **Review round** — failed words from the main round are replayed at the end
- **Perfect round** unlocks themed bonus vocab packs and gives a +20 pt bonus
- **Screen time reward** — 1 study minute = 2.5 minutes of parent-approved screen time
- **Parent zone** — PIN/biometric-protected dashboard with weekly reports, screen time approval, and vocab import

## Stack

- Flutter 3.41+ / Dart 3.11+
- Firebase: Auth (Google/Apple), Firestore
- ML Kit: Digital Ink Recognition (German), Text Recognition (OCR)
- speech_to_text (German locale)
- Riverpod (state) + go_router (navigation) + Hive (local cache)

## Setup

### 1. Install Flutter

Requires Flutter SDK 3.41 or later. See [docs.flutter.dev/get-started](https://docs.flutter.dev/get-started).

### 2. Firebase configuration

The repo ships `lib/firebase_options.dart` with placeholder values so the project compiles. Before running, generate the real config:

```bash
dart pub global activate flutterfire_cli
flutterfire configure --project=<your-firebase-project-id>
```

This will overwrite `lib/firebase_options.dart` and drop `google-services.json` into `android/app/`. Both files are gitignored for safety — **never commit the populated versions**.

### 3. Install dependencies + generate Hive adapters

```bash
flutter pub get
dart run build_runner build --delete-conflicting-outputs
```

### 4. Run

```bash
flutter run                  # debug build on connected device
flutter build apk --release  # release APK
```

## Game rules (canonical)

| Action                  | Points | Dominates word? |
|-------------------------|--------|-----------------|
| Flash card flip         | +1     | No              |
| Keyboard correct        | +2     | No              |
| Handwriting no errors   | +7     | **Yes**         |
| Voice correct           | +7     | Bonus only (on already-dominated word) |
| Review round pass       | +7     | **Yes**         |
| Perfect round bonus     | +20    | —               |

- A lesson reaches `status: completed` when all words are dominated.
- A lesson reaches `status: perfect` when the review round is passed with 0 errors. This unlocks `extraVocabUnlocked` and the voice mode.
- 1 effective study minute → 2.5 minutes of screen time (parent-approved).

## Conventions

- `WoerterAbenteuerApp` (no `ö`) — `ö` is illegal in Dart identifiers.
- Use `withValues(alpha:)` instead of deprecated `withOpacity()` (Flutter 3.41+).
- Orientation is locked to portrait.
- Anthropic API keys, Google Speech keys, and any other secrets MUST live in Cloud Functions — never in client code.
